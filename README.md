# Zia_graduation_pj
Improving Hull Form of a 1756 TEU Container Ship for Better Energy Efficiency
Project Overview

This project focuses on improving the energy efficiency of a 1756 TEU container ship (Tzini) by optimizing its hull form to reduce hydrodynamic resistance and fuel consumption. The study supports environmentally friendly ship design by lowering energy demand, greenhouse gas emissions, and operating costs.

Objectives

* Analyze the characteristics of an existing container ship hull design.
* Modify hull form parameters to improve performance.
* Investigate the influence of hull coefficients on ship resistance.
* Reduce total resistance and effective power requirements.
* Validate the optimized design using computational tools.

Methodology

Hull Form Generation

* Created the ship’s lines plan using AutoCAD.
* Generated and faired the 3D hull model in Maxsurf Modeler Advanced.
* Applied NURBS surface modeling to obtain a smooth and realistic hull form.

Resistance Prediction

* Implemented the Holtrop resistance prediction method in MATLAB.
* Calculated total ship resistance and effective power requirements.
* Optimized hull parameters through iterative computations.

Hull Form Optimization

Key parameters modified during optimization:

* Block Coefficient (CB)
* Prismatic Coefficient (CP)
* Longitudinal Center of Buoyancy (LCB)

Validation

* Verified resistance predictions using SHIPFLOW CFD software.
* Compared performance between the original and optimized hull forms.

Tools & Technologies

* AutoCAD
* Maxsurf Modeler Advanced
* MATLAB
* SHIPFLOW (CFD Analysis)
* Key Achievements

* Reduced total ship resistance.
* Lowered effective power requirements.
* Improved fuel efficiency.
* Enhanced compliance with IMO Energy Efficiency Design Index (EEDI) objectives.
* Developed a more environmentally friendly hull design.

Conclusion

The optimized hull form successfully reduced resistance and power consumption compared to the original Tzini design. The project demonstrates how hull form optimization, supported by MATLAB-based calculations and CFD validation, can contribute to sustainable ship design and improved maritime energy efficiency.
